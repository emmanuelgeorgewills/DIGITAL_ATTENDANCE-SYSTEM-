# Digital Attendance System

## Overview
This is a Digital Attendance System designed for hour-based lectures. It allows for the registration of students, creation of lecture sessions, marking of attendance, and generating reports. It ensures that the attendance records are saved and loaded from text files, providing persistent data between program executions.

## Features
- Add and view students.
- Create lecture sessions.
- Mark student attendance (Present, Absent, Late).
- View attendance for a session.
- Save and load data from text files.

## Files
- `main.cpp`: Contains the main logic for student and attendance management.
- `students.csv`: Stores registered student data.
- Session files: Stores attendance for each lecture session (e.g., `session_EE201_2026_02_23.csv`).

## How to Use
1. Add students using the menu option.
2. Create a lecture session.
3. Mark attendance for the students.
4. View the attendance summary and session records.

## Requirements
- C++ Compiler (e.g., GCC)
- Text editor (e.g., Visual Studio Code)
